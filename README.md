## ReactJs E-Commerce
Bir ReactJs e-ticaret uygulaması, 'dummyjson' api servisi kullanan bir frontend projesi.

## Server Rest Api
- https://dummyjson.com/docs

## Kullanılan Teknolojiler
- ReactJS
- React Router Dom
- Axios
- Crypto-js
- Simple Image Slider
- Simple Star Rating
- Toastify
- Image Gallery
